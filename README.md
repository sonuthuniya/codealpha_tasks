# 🧮 Simple Calculator in C

A beginner-friendly calculator program written in **C language**.  
It performs four basic arithmetic operations: **Addition, Subtraction, Multiplication, and Division** with proper error handling.

---

## 🎯 Objectives
- To understand **input/output handling** in C.
- To practice **switch-case statements** for decision making.
- To implement **error handling** (division by zero).
- To build a **command-line calculator** for practical learning.

---

## 📌 Features
- Accepts two numbers as input.
- Supports operators: `+`, `-`, `*`, `/`.
- Handles division by zero gracefully.
- Displays results with **two decimal precision**.
